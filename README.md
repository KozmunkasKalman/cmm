# C++ (ejtsd: „cé meg meg”)
Ezen C++ könyvtár lehetővé teszi, hogy ötvözd a C++-t és az anyanyelved, hogy ezáltal leghatékonyabban tudj számítógépeden munkaterveket (programokat) alkotni. Mellesleg itt a „C” nem a C/C++ programozási nyelvekhez kapcsolódik, hanem a cigányokhoz.

## Használat
1. Töltsd le a "cé_meg_meg.h" fájt, és tedd a forráskóddal egy helyre
2. A kódnak a következő sorral kell hogy kezdődjön: ``include "cé_meg_meg.h"``

## Főbb szintaktikai elemek:
### 1. Adattípusok:
- void => űr
- const [...] => állandó/konstans [...]
- int => egész
- float => tört/lebegő
- size_t => méret
- char => karakter
- bool => logért/logikai/igazhamis

- std::vector => szabvány::vektor
- std::string => szabvány::fonal/karakterfonal/szövegfonal/fonál/karakterfonál/szövegfonál/zsinor/karakterzsinor/szövegzsinor/szöveg
- std::fstream => szabvány::fájlpatak

### 2. Visszatérés és ki-/bemenet
- return => visszatér

- std::cin => szabvány::bemenet
- std::cout => szabvány::kimenet
- std::cerr => szabvány::hiba
- std::endl => szabvány::sorvég

### 3. Beépített funkciók
- exit() => kijárat()
- rand() => véletlenszerű()
- system() => rendszer()/szisztéma()

### 4. Állítások és ciklusok
- if => hogyha
- else if => különben ha
- else => különben

- while => amíg
- for => merthogy

- break => törés
- continue => folytatás

- switch => választás
- case => eset
